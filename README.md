Trader Performance vs Market Sentiment

Objective
Analyze how Bitcoin Fear/Greed sentiment affects trader performance on Hyperliquid.

Data Preparation
- Cleaned and merged trader data with daily sentiment
- Converted timestamps to daily level
- Created performance metrics (PnL, win rate, position size)

Key Insights
1. Extreme Greed periods show highest profitability and win rate.
2. Extreme Fear periods show weakest performance.
3. Trader position sizing behavior changes across sentiment cycles.

Strategy Recommendations
- Increase exposure during Extreme Greed periods.
- Reduce leverage and risk during Extreme Fear periods.

Tools Used
Python, Pandas, Seaborn, Matplotlib
